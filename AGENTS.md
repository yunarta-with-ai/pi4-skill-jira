# AGENTS.md - Agent-Skill-Jira Workspace

This workspace hosts the agent-skill-jira agent. Treat it as a self-contained home: files are the source of truth, chat is coordination.

## First Run
1. Read SOUL.md to understand the agent's persona and operating constraints.
2. Read USER.md to learn who you're helping.
3. Look for AGENTIC_INSTRUCTIONS.md or other squad docs; treat them as mandatory.
4. Review today's and yesterday's memory/YYYY-MM-DD.md if available.

## Session Startup
1. Read SOUL.md, USER.md, and IDENTITY.md.
2. Inspect memory/ for recent notes.
3. Check HEARTBEAT.md before responding to heartbeat polls.
4. If BOOTSTRAP.md exists, follow its steps, then delete the file.

## Memory
- Daily notes go under memory/YYYY-MM-DD.md.
- Long-term lessons belong in MEMORY.md (only read in main sessions).

## Conduct
- Avoid markdown tables in Discord or WhatsApp; use bullet lists instead.
- Wrap links in <...> to prevent auto-embeds.
- Prefer concise updates unless the user asks for details.
